# react-stardict
词库是基于[ECDICT](https://github.com/skywind3000/ECDICT)，感谢skymind3000整理了大约340万词条的词库。
后端是使用了express，sqlite3和httpProxy。前端是使用react和create-react-app构建的。
在项目中，让我对react熟悉不少，也自己手写了UI组件，也是恶补了一波css。如果此仓库对你有帮助的话，欢迎star。
